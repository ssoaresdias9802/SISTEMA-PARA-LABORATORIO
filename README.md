## 📌 About the Project

This project is a **Laboratory Management System** developed in Java. It simulates the workflow of a clinical lab, allowing the registration of patients, management of requested exams, report generation, and statistical analysis.

The goal is to provide a structured and efficient way to handle daily laboratory operations using core programming concepts.

---

## ⚙️ Features

* 📥 Load available exams from file
* 👤 Register new patients with requested exams
* 🔍 Search patients by CPF
* 📄 Generate daily reports (`.txt`)
* 📊 View statistics of exams performed
* 🗂️ Custom implementation of linear list

---

## 🧠 Concepts Applied

* Object-Oriented Programming (OOP)
* File I/O (BufferedReader, FileWriter)
* Data Structures (custom linear list)
* Exception Handling
* Date manipulation with `LocalDate`
